Work Experience

Data Breach Review

Comprehensive assessment conducted after a security incident where unauthorized access to sensitive information occurs. This review involves analyzing the breach, conducting forensic investigations, assessing the impact, notifying affected parties, ensuring legal compliance, and implementing corrective measures to prevent future breaches. The process aims to understand the breach's scope, mitigate damage, comply with regulations, and improve overall cybersecurity.

Software Development

Systematic process involving requirements gathering, design, coding, testing, deployment, and maintenance. It requires collaboration, project management, version control, and consideration of security measures. Developers use various methodologies like Agile and DevOps, and ongoing learning is essential to stay updated with evolving technologies. Data Annotation

Process of labeling data with metadata to enable machine learning models to understand and learn from it. This involves techniques such as bounding boxes, segmentation, and classification across various data types like images, text, video, and audio. Common use cases include object detection, NLP tasks, autonomous vehicles, and medical imaging. Challenges include subjectivity and scalability, and various tools and platforms are used to facilitate the annotation process.
